介绍

Spock 框架参考文档

Peter Niederwieser，Leonard Brünings，Spock 框架团队 - 2.0版

Spock 是用于 Java 和 Groovy 应用程序的测试和规范框架。使它从人群中脱颖而出的是其美丽且极具表现力的规范语言。 由于它的 JUnit 运行器，Spock 与大多数 IDE、构建工具和持续集成服务器兼容。 Spock 的灵感来自 JUnit 、 jMock 、RSpec 、Groovy 、 Scala 、Vulcans 和其他迷人的生物们。
