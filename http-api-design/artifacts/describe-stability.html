描述稳定性

描述稳定性

根据 API 的成熟度和稳定性来描述您的 API 或其各种端点的稳定性，例如使用原型/开发/生产标志。

有关 可能的稳定性和变更管理方法，请参阅Heroku API 兼容性政策 。

一旦您的 API 被声明为生产就绪且稳定，请不要在该 API 版本中进行向后不兼容的更改。如果您需要进行向后不兼容的更改，请创建一个版本号递增的新 API。
